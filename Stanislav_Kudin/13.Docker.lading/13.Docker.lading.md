```bash
CMD ["python3", "/listener/socket_listen.py"]


EXPOSE 8859
```



# socket_listen.py
```python

#!/usr/bin/env python3

import socket

PORT = 8859

with socket.socket(socket.AF_INET, socket.SOCK_STREAM) as s:
    s.bind(("", PORT))
    s.listen()
    conn, addr = s.accept()
    with conn:
        print('Connected by', addr)
        while True:
            data = conn.recv(1024)
            if not data:
                break
            conn.sendall(data)
```
