## add and setup user 

```bash
adduser mataras
sudo usermod -aG sudo mataras
echo 'mataras ALL=(ALL) NOPASSWD: ALL' | sudo EDITOR='tee -a' visudo

```

## /etc/ansible/hosts

```bash

[infra:children]
internal

[jump]
bastion ansible_host=192.168.1.70

[internal]
centos ansible_host=10.20.30.11
ubuntu ansible_host=10.20.30.12

```
